# ExamineAssistant

Невидимый помощник для созвонов на macOS. Слушает собеседника и микрофон, по хоткею отвечает от твоего лица голосом Claude. Окно не видно при демонстрации экрана.

**Стек:** PyQt6 · OpenAI `gpt-4o-transcribe` · Claude `sonnet-4-6` · ScreenCaptureKit

Системный звук (голос собеседника) захватывается через **ScreenCaptureKit** — нативный API macOS. **BlackHole не нужен.** Если ScreenCaptureKit недоступен (macOS < 13), приложение откатится на BlackHole.

## Установка

```bash
brew install uv     # пакетный менеджер (быстрый)
make install        # зависимости через uv + .env
```

Если `uv` не поставлен — `make install` откатится на pip (медленнее).

Впиши ключи в `.env`:
```
OPENAI_API_KEY=sk-...
ANTHROPIC_API_KEY=sk-ant-...
```

В **Системных настройках → Конфиденциальность** дай Terminal'у (или собранному `.app`):
- **Запись экрана** — чтобы слышать системный звук собеседника
- **Микрофон** — чтобы писать твой голос
- **Универсальный доступ** — чтобы работали глобальные хоткеи

После выдачи разрешений — полностью закрой Terminal (⌘Q) и открой заново.

## Запуск

```bash
make run
```

Появится тёмное окно поверх всех. Не показывается при screen capture в Zoom/Meet/Teams.

## Хоткеи

| | |
|---|---|
| **⌘R** / **⌘К** | Ответить на последние 60 сек разговора |
| **⌘Y** / **⌘Н** | Открыть поле для своей заметки (⌘↵ — отправить, Esc — отмена) |
| **⌘E** / **⌘У** | Очистить буфер и историю |
| **⌘H** / **⌘Р** | Скрыть / показать окно |

Работают глобально — даже когда фокус в Zoom. Русская раскладка зеркальна.

## Профили

В шапке окна — выпадающий список. По умолчанию: **Общий**, **Учёба**, **IT собеседование**.

- **✎** — отредактировать текущий профиль
- **＋** — создать новый

Профили хранятся в `profiles/*.yaml`. Каждый — это system prompt для Claude (кто я, по какому предмету, как должен отвечать).

## Команды

```bash
make help       # список команд
make install    # синхронизировать .venv с uv.lock
make run        # запустить десктоп
make cli        # запустить CLI-версию (отладка)
make lock       # обновить uv.lock из pyproject.toml (после правки зависимостей)
make sync       # синхронизировать .venv (то же что install)
make clean      # удалить кэш
make reset      # снести venv (потом make install заново)
```

## Структура

```
app/core/     бизнес-логика: audio, transcribe, claude, profiles, billing
app/ui/       PyQt6 интерфейс: окно, контроллер, worker в QThread
profiles/     YAML с предметами
main.py       десктоп-приложение
cli.py        CLI-версия (без UI)
```

## Стоимость

Внизу окна — реалтайм-счётчик в рублях по фактическим API-ценам:
- `gpt-4o-transcribe`: $0.006/мин
- Claude Sonnet 4.6: $3 / $15 за миллион токенов input/output

Без наценок. Один час активного использования — ~50-100₽.
